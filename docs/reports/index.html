Founder workflow

Daily glance (~3 min): Dashboard. Skim metric cards, check budget bar, scan recent handoffs.

Weekly review (~20-30 min): Proposals. Mark approve/reject/defer, export JSON, run .claude/scripts/apply-decisions.sh. Read latest weekly report.

Per-ship review (15-30 min): Ship report when closed. Discussion Bubbles for contested decisions, activity for agent conversation trace.

Per-wave review (1-2 hours): Wave report, decision dissent, proposal impact.
