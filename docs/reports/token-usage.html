How estimation works: when a cron session invokes Claude Code, the session's start+end timestamps from the log file give a duration in seconds. We estimate input tokens at duration_seconds * 100 and output tokens at duration_seconds * 30. Constants live at the top of scripts/aggregate-token-usage.py as INPUT_RATE_TOKENS_PER_SEC / OUTPUT_RATE_TOKENS_PER_SEC - recalibrate against the first Founder manual paste if needed.
