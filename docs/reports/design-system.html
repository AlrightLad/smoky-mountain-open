PARBAUGHS DESIGN SYSTEM · PHASE 1 · BILLIARD GREEN + CHALK + BRASS

A country club after hours, run by people who know the game. Premium without precious. Brass scarcity is the brand.

Three colors do the real work. Type comes from a system stack with intentional weight. Motion serves comprehension. Every token here has a stated purpose and a stated anti-use. The round-trip test enforces the rules the brief sets.

Anti-patterns — what NOT to do

Critic gates UI work against these. If any apply on a PR, the work returns for revision.

Anti-pattern · brass for generic CTA
Don't use brass for primary actions that aren't achievement-related.
"Log a round" is chalk-on-green. "Claim wager winnings" is brass-accented. The two are different signals — confusing them flattens the brand. Brass scarcity is what makes brass mean something.
Anti-pattern · accent overload
Don't combine more than 2 accent colors per screen.
A claret error + a brass champion + a moss success + a teal info = the screen is doing four things. The eye picks none. Pick the signal that matters most; demote the rest to neutral or remove.
Anti-pattern · display below 768px
Don't use --text-display (48px) below 768px viewport.
Display text on small viewports overruns lines and feels shouty. Step down to --text-2xl or --text-xl on mobile via a media query.
Anti-pattern · shadows on small cards
Don't use --shadow-lg on small cards.
--shadow-lg communicates "this is floating high above the page" — modals, popovers. A small inline card with --shadow-lg reads heavy. Lift small cards with tonal contrast (raised green) instead.
Anti-pattern · decorative motion
Don't animate things that aren't communicating state.
A heart pulse on the "like" button is state. A logo that breathes on the loading screen is decoration. The first stays; the second goes.
Anti-pattern · emojis where SVG icons exist
Don't use system emoji as iconography (W1.S1 LB1, locked).
Emojis render differently on every OS, break the editorial voice, and are unaccessible without explicit aria-labels. Use the SVG icon library. The one allowed exception is ⛳ on the Caddy bot — that's been Founder-ratified as a brand signature.
Anti-pattern · raw hex / px values in source
Don't put raw hex, px, or ms values in any file outside design-tokens.css.
If the value you need doesn't have a token, the work STOPS and a token-add proposal is authored first. The round-trip test fails the build if a raw value slips through. Discipline that isn't enforced by the toolchain doesn't survive.
