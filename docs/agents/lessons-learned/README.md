# Lessons Learned

Per-ship and per-wave lessons captured during retrospectives.

## File naming

- Per-ship: `SHIP_<ID>_LESSONS.md` (only if ship surfaced ship-specific lessons)
- Per-wave: `WAVE_N_LESSONS.md` (aggregated at wave close)

## Capture protocol

Per RETROSPECTIVE_REVIEW.md:

1. Concrete observation
2. Root cause
3. Concrete change to skill, hook, protocol, or governance doc

Lessons that warrant systemic change result in updates to skills, hooks, or governance documents — not just lesson capture.
