## Vision (ratified — DO NOT MODIFY)

W1.I5 ships the crisis banner system — 3-tier (NOTICE / ALERT / CRITICAL) member-visible banner controlled via Founder admin tooling + admin SDK script for emergency activation when admin tooling itself is impaired. Member-dismissible per session for NOTICE; ALERT auto-clears on resolution; CRITICAL requires Founder dismissal. Per locked governance: orchestration team can deactivate ALERT/NOTICE autonomously but CRITICAL needs Founder. Per locked Vision discipline: deep research; fault-tolerant revert if banner pattern breaks page chrome.
