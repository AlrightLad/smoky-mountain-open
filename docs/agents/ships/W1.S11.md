## Vision (ratified — DO NOT MODIFY)

[Per W1.S11_VISION_AMENDMENT.md — see governance docs] The Feed + Activity ship lands the platform's social communication and activity surfaces unified under one architecture — Feed cards (round posts, Chip posts, auto-posted activity events) + League Pulse activity stream + Activity detail view. Auto-posted activity events: round finalized, championship awarded, friend joined, wager initiated / settled, party game claimed, course added, Trophy earned.
