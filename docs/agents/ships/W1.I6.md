## Vision (ratified — DO NOT MODIFY)

W1.I6 ships course infrastructure foundation — members upload scorecard photo (mobile via Capacitor camera, web via file upload, both surfaces from Day 1), free-tier OCR (Tesseract.js client-side, no third-party API) extracts ALL tee box configurations in single upload (2D matrix holes × tee boxes), member reviews and corrects extracted data, CourseDoc created in Firestore. Duplicate detection blocks + surfaces existing course with prompt 'did you mean to re-upload?' Single-approver community verification (any member approves), provisional-playable rounds at unverified courses, broadcast to every league that's played the course on approval. Per locked Vision discipline: deep research; fault-tolerant revert if OCR pattern proves accuracy-insufficient.
