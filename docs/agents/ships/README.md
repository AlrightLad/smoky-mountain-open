# Active Ships

This directory contains Ship Plans for ships currently in progress.

One file per active ship, named by ship ID: `W1.S1.md`, `M3.md`, `AS1.md`, etc.

Use [`../SHIP_PLAN_TEMPLATE.md`](../SHIP_PLAN_TEMPLATE.md) for new ships.

On ship close, file moves to `../ship-reports/` per Orchestrator close-out protocol.
