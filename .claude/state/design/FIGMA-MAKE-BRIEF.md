Design a cohesive SET of 5 premium collectible AVATAR RING FRAMES for "PARBAUGHS," an

5 tiers, each visually DISTINCT (no two alike), rarity-laddered:
1. Common — a clean struck antique-brass band, fine beaded edge, restrained.

3. Rare — brass + a repeating crossed-hickory-clubs & claret-pennant motif (the club band).
4. Epic — brass wreathed with the claret-rose + green-thorn vine (the Parbaughs rose).
5. Legendary — a gallery rope-and-stanchion brass ring with a claret enamel crest at top.

Each: perfectly circular + symmetrical, head-on flat view, large hollow transparent center,
jeweler-grade soft highlights + gentle relief, premium + dignified, NO text, NO cartoon,
NOT a gold medal, NOT a Roman laurel. Export each as a transparent PNG, 1:1, centered.
