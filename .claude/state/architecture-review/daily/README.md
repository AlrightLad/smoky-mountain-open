# Daily architecture-review output

One file per day: `<YYYY-MM-DD>.md`. Generated by the Architecture
agent's daily cycle (30-60 min runtime).

## Required sections

```markdown
# Daily health — <YYYY-MM-DD>

## Substrate state
(amendments active, proposals in flight, escalations open)

## Agent activity rollup
(commits per agent in last 24h, queue depth per agent,
 in-progress tasks)

## Stop-decision pattern check
(any false-stops? real-stops vs Q1.A-G? recurring conditions?)

## Error-backlog severity
(CRITICAL / HIGH / MEDIUM / LOW counts;
 delta vs yesterday)

## Daily priorities
(top 3 things to watch today)

## Banner data emitted
(aggregates/architecture-review.json updated → ...)
```

Output also gets aggregated into `weekly/<YYYY-WW>.md` and
`monthly/<YYYY-MM>.md` over time.
